# MTFC Scenario Phase

Scenario-phase modeling code used for the Modeling the Future Challenge workflow.

This repository is intentionally scoped: it captures the assumptions, transformations, and outputs used in the scenario phase so the work can be audited and reproduced.

---

## What this does

- Encodes scenario assumptions and parameter choices
- Produces scenario outputs (tables/figures) from the same inputs deterministically
- Keeps the scenario phase cleanly separated from exploratory analysis
